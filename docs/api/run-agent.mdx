# runAgent()

Execute an AI agent with a natural language instruction. This is the primary interface of the OpenQA agent harness.

## Signature

```typescript
runAgent(
  provider: ProviderConfig,
  prompt: string,
  pageOrContext: Page | BrowserContext,
  options?: RunAgentOptions
): Promise<string>
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | `ProviderConfig` | Yes | Agent provider, created with `claudeCode()` |
| `prompt` | `string` | Yes | Natural language instruction |
| `pageOrContext` | `Page \| BrowserContext` | Yes | Playwright page or browser context |
| `options` | `object` | No | Configuration options |

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `verbose` | `boolean` | `true` | Enable console logging |
| `returnUsage` | `boolean` | `false` | Return token usage statistics |

## Returns

- **Default**: `Promise<string>` — Final agent response text
- **With `returnUsage: true`**: `Promise<{ result: string, usage: object, steps: number, sessionId: string }>` 

## Examples

### Basic Usage

```typescript
import { test } from "@playwright/test";
import { runAgent, claudeCode } from "openqa";

test("AI fills form", async ({ page }) => {
  await page.goto("https://example.com");
  await runAgent(claudeCode('claude-haiku-4-5'), "Fill in the contact form with test data", page);
});
```

### In a BDD Step Definition

```typescript
import { runAgent, claudeCode } from 'openqa';
import { aistep } from './fixtures';

aistep(/^(.*)$/, async ({ page }, action: string) => {
  await runAgent(claudeCode('claude-haiku-4-5'), action, page, { verbose: true });
});
```

### With Token Usage

```typescript
const result = await runAgent(
  claudeCode('claude-sonnet-4-6'),
  "Complete the checkout flow",
  page,
  { returnUsage: true }
);
console.log(result.usage);
```

## claudeCode()

Creates a Claude Code provider configuration for use with `runAgent`.

```typescript
import { claudeCode } from 'openqa';

const provider = claudeCode('claude-haiku-4-5');  // default model
const provider = claudeCode('claude-sonnet-4-6');
const provider = claudeCode('claude-opus-4-7');
```

| Model | Description |
|-------|-------------|
| `claude-haiku-4-5` | Fast, cost-efficient (default) |
| `claude-sonnet-4-6` | Balanced performance |
| `claude-opus-4-7` | Most capable |

## Session Management

The harness maintains a per-`BrowserContext` Claude Code conversation session. Across multiple `runAgent()` calls within the same Playwright test, Claude resumes its conversation, giving it context about earlier steps.

Reset the session manually if needed:

```typescript
runAgent.resetSession(context);
```
